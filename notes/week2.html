Mongo CRUD functions

  • C .insertOne() - Used to Create a new item in the collection
  • R .find() - Used to Read (get) an item in the collection
  • U .replaceOne() - Used to Update an item in the collection
  • D .deleteOne() - Used to Delete an item in the collection
