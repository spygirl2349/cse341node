Github Shortcuts

  • git add . will add the changes to prep to commit it.
  • git commit -m "message here" commits the changes.
  • Lastly, git push pushes the changes to the Github repo.

Other Helpful Shortcuts

  • CTRL + C = stop web server (for node.js listening)
  • CTRL + / = comment out code # this is a python comment
