Github Shortcuts

  • git add . will add the changes to prep to commit it.
  • git commit -m "message here" commits the changes.
  • Lastly, git push pushes the changes to the Github repo.

Other Helpful Shortcuts

  • CTRL + C = stop web server (for node.js listening)
  • CTRL + / = comment out code # this is a python comment
  • ALT + Z for word wrap in VS Code

Node package (Express)

Make sure that you have express installed. Install express by:

  • In the terminal, under the desired folder put: npm init --y
    • This adds a package.json file to the folder
    • Paste npm install --save express into the terminal to add the "node_modules" folder and package-lock.json file

Nodemon

Nodemon is a dev tool to help the project reload automatically after saving so you don't have deploy and non deploy over and over again.

MongoDB

Packages needed to connect to MongoDB:

  • npm install mongoose
  • npm install mongodb
  • npm install dotenv I'm not sure if this has everything to do with MongoDB, but it is important to make sure that your .env file actually works and can connect with the rest your code.

Connect to a MongoDB with Node.js - video is alright, but the article has great code that you can refer to and copy.

Render

Make sure you add the .env file information in the "Environment" tab when connecting to VS Code / Github because the .env isn't synced throught Github for privacy.

Under Settings, make sure that the Start Command has a command that will match yours. For example, if you have "node server.js" there needs to be a file called server.js in your program.
